🎥 Object Detection and Tracking System

📌 Internship Project – Task 4

This project is an AI-powered Object Detection and Tracking System developed as part of my internship assignment.
The system detects and tracks objects in real-time video using computer vision and deep learning techniques.

🚀 Project Overview

The system enables:

Real-time object detection using pre-trained models
Tracking objects across video frames
Assigning unique IDs to each object
Displaying bounding boxes and labels

It uses models like YOLO for detection and tracking algorithms such as SORT or Deep SORT.

✨ Features
🎥 Real-time video processing
🧠 Object detection using YOLO
🔄 Object tracking with unique IDs
📦 Bounding box visualization
⚡ Fast and efficient performance
🖥️ Works with webcam or video file
🛠️ Tech Stack
Language: Python
Libraries: OpenCV, NumPy, PyTorch
Model: YOLO (v5/v8)
Tracking: SORT / Deep SORT
⚙️ How It Works
Capture video from webcam or file
Process each frame using OpenCV
Detect objects using YOLO model
Track objects across frames
Assign unique tracking IDs
Display results in real time
